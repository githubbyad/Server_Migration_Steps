  21. Hit CounterIf hit counter is showing form third-party site, then we need to covert it into system's hit counter.

    Below are the steps to follow.

    NOTE There are two "C" steps added as we have placed different counters on some sites.

    A Submit "Settings-Staff" Screen. This will update the code of the new Hit Counter.

    B Now we don't have the latest counter number to set. So Open "Settings-Staff" screen and look for the "Hit Counter Code" field.

    C If the the code under this field shows from "webfreecounter.com" site, then you need to open below URL,
                https://www.webfreecounter.com/hit.php?id=XYZ
          Where XYZ is the id to get. See screenshot: https://i.snipboard.io/PhYeIt.jpg
          Once you visit the site, you will get the latest counter number. Save that counter number.
         NOTE Take backup of code added in "Hit Counter Code" field from "Settings-Staff" and clear that code from this field.

    OR

    C On some sites, we have directly placed third-party code under "Free Layout". So you can get the latest number after visiting the site. Save that counter number. Now visit "Free Layout" and replace third-party code with {{PutHitCounter}} command. Make sure to take backup of third-party code.

    D Now run below script and make sure to change site URL where you will see newspaperpal.com
                /staff_reset_site_counter.asp?site=newspaperpal.com&sm=Y
          Place the latest counter number and submit.

    E Visit the site and you will see a hit counter in text format having comas.

    Make sure to visit all pages and flag to Mam if it requires to update pages.
    In case you face any issue, please contact Adnan.
